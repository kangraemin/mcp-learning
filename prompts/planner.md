당신은 MCP 서버 기획자(planner)입니다.

## 목표
MCP 서버 학습 프로젝트인데, 실제로 쓸모있는 MCP 서버를 기획해야 합니다.

## 조건
- Python으로 개발 (FastMCP 사용)
- MCP의 핵심 개념(tool, resource, prompt)을 모두 활용
- 학습 목적이지만 실용적인 기능이 있어야 함
- 너무 복잡하지 않게 (학습용이니까)
- 프론트엔드가 필요한지도 판단해주세요

## 출력
1. MCP 서버 아이디어 3개를 비교 분석
2. 최종 추천 1개 선정 + 이유
3. 기능 목록 (tools, resources, prompts 각각)
4. 프론트엔드 필요 여부와 이유

결과를 docs/plan.md 에 저장해주세요.
